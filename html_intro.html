HTMLの役割 — Webページの「骨組み」

HTML（HyperText Markup Language）は、Webページの構造を定義する言語です。文章を見出し・段落・リスト・リンク・画像などに分け、ブラウザに「どう表示するか」を伝えます。デザイン（色や配置）は主にCSS、動きを付けるのはJavaScriptが担いますが、すべての基礎はHTMLです。

よく使う要素とその意味

  • <h1>〜<h6> — 見出し（重要度により数字が小さいほど上位）
  • <p> — 段落（本文）
  • <a href="…" > — リンク
  • <img src="…" alt="…" > — 画像（altは必ず）
  • <ul> / <ol> / <li> — リスト（順不同／有序）
  • <header> / <main> / <footer> — ページ構造を明確にするセクショナル要素

簡単なHTMLの例（実際に動くサンプル）

<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <title>サンプルページ</title>
</head>
<body>
  <h1>こんにちは、HTML！</h1>
  <p>これはサンプルの段落です。</p>
</body>
</html>

アクセシビリティとSEOの基本ポイント

- alt属性：画像に必ず意味のある代替テキストを入れる。
- 見出しの順序：h1→h2→h3のように論理的に使う。
- メタデータ：titleとmeta descriptionは検索結果に直結するので適切に設定する。

よくあるつまずきと対処法

- 表示が崩れる：CSSのクラス名やセレクタが正しいか、パス（リンクや画像）が合っているか確認。
- 文字化け：meta charsetがutf-8になっているかをチェック。
- リンク切れ：相対パス/絶対パスの違いを理解する。

まとめ（最初に覚えるべきこと）

  1. DOCTYPEとhtml langを設定する
  2. titleとmeta descriptionを入れる
  3. 見出しと段落で文章を整理する
