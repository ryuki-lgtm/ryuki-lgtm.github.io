CSSとは何か？

CSS（Cascading Style Sheets）は、HTMLで作った構造に「見た目」を与えるためのルールです。色、余白、フォント、配置などを指定して、ページを見やすく整えます。HTMLは骨組み、CSSは服やデザイン、JavaScriptは動き、というイメージです。

基本の書き方（外部スタイルシート）

HTML の <head> に以下のようにリンクします：

<link rel="stylesheet" href="style.css">

よく使うプロパティ

  • color / background-color：文字色と背景色
  • padding / margin：内側と外側の余白
  • font-size / line-height：文字の大きさと行間
  • display: block / inline / flex：表示方法（特にflexはレイアウトで強力）

ボックスモデルの理解

要素は「コンテンツ → padding → border → margin」の順で外側にスペースを持ちます。レイアウトが崩れる場合、まずこのボックスモデルのどこが影響しているかをチェックしましょう。

簡単なレイアウト例（フレックス）

.container {
  display: flex;
  gap: 16px;
}
.item {
  flex: 1;
}

レスポンシブ対応の基礎

- meta viewport を入れる。
- 幅指定は固定(px)より%やmax-widthを使う。
- メディアクエリで画面幅に応じてスタイルを切り替える。

よくある失敗と改善

1. クラス名を適当に付けると後で管理が大変 → 命名規則（BEMなど）を検討。
2. グローバルなリセットがないとブラウザ差が出る → 最低限のリセットを入れる。
3. 画像や長いテキストで崩れる → max-widthやword-breakを指定。
