JavaScriptで何ができるのか？

JavaScriptはWebページに動きを付けるための言語です。フォームの入力チェック、クリックでの表示切替、外部API からのデータ取得など、ユーザー操作に応じてページを変化させることができます。

基本の構文（例）

<script>
const btn = document.getElementById('btn');
btn.addEventListener('click', () => {
  alert('クリックされました');
});
</script>

簡単な実例：ボタンでテキストを切り替える

以下のHTMLをページに入れて動かしてみてください：

<button id="toggle">切替</button>
<p id="text">表示A</p>

<script>
document.getElementById('toggle').addEventListener('click', function(){
  const p = document.getElementById('text');
  p.textContent = p.textContent === '表示A' ? '表示B' : '表示A';
});
</script>

イベントとDOMの基礎

- イベントリスナー：ユーザーの操作をキャッチして処理を実行。
- DOM操作：要素の取得（getElementById / querySelector）と内容や属性の変更。
- 非同期処理：fetchで外部データを取得する際に使う（後述のAPI利用）。

注意点（初心者が陥りやすい）

  • スクリプトはDOMが読み込まれた後で実行する（bodyの最後に置く、またはDOMContentLoadedで待つ）。
  • コンソール（F12）を活用してエラーを確認する習慣をつける。
  • 初心者はグローバル変数を多用しがち → スコープを意識する。

まとめと学習の次の一歩

実際に手を動かして、小さな機能（モーダル、画像の切替、簡単な計算機）を作ることが上達の近道です。作った成果は静的サイトにまとめておくと、ポートフォリオにもなります。
