GitHub Pagesで無料公開する方法

ホーム | HTMLとは | CSSの基本 | JavaScriptの始め方 | プライバシーポリシー

前提：無料で手軽に公開したい人向け

GitHub Pagesは、静的ファイル（HTML/CSS/JS）を無料でホストできるサービスです。独自ドメインも設定可能で、初心者が試すには最適。ここでは手順を簡潔にまとめます。

ステップ1：GitHubアカウントを作る

GitHub にアクセスしてアカウント登録を行います（ユーザー名とメールが必要）。

ステップ2：新しいリポジトリを作る

  1. リポジトリ名を指定（例：yourname.github.io または 任意名）
  2. 公開（Public）にする
  3. READMEを追加しておくと親切

ステップ3：ファイルをアップロード

- ブラウザ上からでもアップロード可能。`index.html`, `style.css`, その他記事ファイルをルートに置く。
- リポジトリが `yourname.github.io` 形式なら、https://yourname.github.io/ で公開されます。

ステップ4：Pages設定（リポジトリ設定 → Pages）

- Branch を `main`（または `master`）に設定し、rootを選択。数分で反映されます。反映されない場合は数分待つかキャッシュをクリア。

独自ドメインを使いたいとき

- ドメインプロバイダ側でCNAME / Aレコードを設定します。GitHubのドキュメントに従ってください。独自ドメインはSEOやブランディングに有利です。

公開後にやるべきこと（AdSense準備）

  • プライバシーポリシーページを必ず用意する
  • サイトの主要ページ（5本以上）を公開してからAdSense申請
  • ページごとにmeta descriptionを整える

手順で不明点があれば、この後すぐに公開作業まで代行します。希望あれば言ってください。
